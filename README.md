Basic Calculator in C

This is a basic command-line calculator program written in C.  
It performs the following operations:
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)

How to Use
1. Run the program.
2. Enter an operator (`+`, `-`, `*`, `/`).
3. Enter two numbers.
4. See the result.
